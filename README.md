**Cesvi** is an interactive educational platform that transforms complex Computer Science concepts into intuitive, visual, and animated learning experiences.

Instead of relying on long textual explanations, **Cesvi** teaches through storytelling, visualization, and step-by-step causal animations that allow learners to observe *why* each concept exists before learning its terminology.

The project is designed for beginners who have little or no Computer Science background, while remaining technically accurate enough to serve as a foundation for more advanced learners.

<br/>

## Vision

Computer Science is often taught through definitions, memorization, and static diagrams. **Cesvi** aims to change that.

Every chapter begins with a real-world observation or question, then gradually reveals the underlying mechanisms through interactive visualizations. Learners experience the problem first, observe the system's behavior, and only then encounter the technical terminology.

This follows a core educational principle:

> **Observation → Question → Mechanism → Terminology**

rather than the traditional approach:

> **Definition → Explanation → Memorization**

<br/>

## Educational Philosophy

**Cesvi** is built around several core principles:

- Learn through visual reasoning instead of memorization.
- Introduce concepts only after learners experience the need for them.
- Separate narrative storytelling from technical mechanisms.
- Preserve conceptual accuracy without overwhelming implementation details.
- Present Computer Science as an interconnected system rather than isolated subjects.

<br/>

## Target Audience

**Cesvi** is designed for:

- Beginners learning Computer Science for the first time.
- High school and university students.
- Self-learners.
- Teachers seeking interactive teaching materials.
- Anyone curious about how computers actually work.

No prior Computer Science knowledge is assumed.

<br/>

## Project Goals

The long-term goal of **Cesvi** is to build a comprehensive visual knowledge base covering the major areas of Computer Science.

Rather than functioning as a traditional textbook, **Cesvi** aims to become an interactive educational experience where learners can explore complex systems through intuitive animations, guided storytelling, and accurate conceptual models.

<br/>

## Guiding Principle

> **Don't memorize how computers work. Watch them work.**
