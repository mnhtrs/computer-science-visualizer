# Cesvi Documentation & Manifest

**What is Cesvi?**
Cesvi is a long-term interactive educational platform that teaches Computer Science, Information Technology, Operating Systems, Networks, Databases, AI, Mathematics for Computing, and any future computational discipline through scientifically correct, interactive visualization. It is not limited to programming or any single subject.

**Repository Organization**
This repository is partitioned into strict domains based on mutability:

- `docs/protocols/` — AI operating law. Read first.
- `docs/constitution/` — Immutable, frozen educational and project principles.
- `docs/governance/` — Immutable rules for project operation and workflows.
- `docs/architecture/` — Frozen technical platform specifications.
- `docs/state/` — Mutable ledger of current project progress.
- `docs/templates/` — Reusable document scaffolds.
- `src/content/` — Educational content modules (Journeys).
- `src/engine/` & `src/rendering/` — The executable software engine.

**Where to Start**
- AI agents: read `docs/README_FOR_AI.md`
- Human contributors: navigate based on the domains above.
